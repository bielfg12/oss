Como Funciona CGNAT?

CGNAT ou Carrier Grade NAT

é uma técnica utilizada para compartilhar um único endereço IP público entre múltiplos usuários, resolvendo a falta de endereços IPv4

Basicamente, o CGNAT aplica o NAT (Network Address Translation) diretamente na rede do provedor, permitindo que vários dispositivos usem o mesmo endereço IP público, mas com portas lógicas diferentes, como se fossem IPs diferentes.

Em resumo o CGNAT funciona como um NAT em larga escala, onde o provedor usa um único endereço IP público para vários usuários, direcionando o tráfego através de diferentes porta
